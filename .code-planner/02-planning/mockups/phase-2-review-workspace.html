RAG

concept aliases: Retrieval-Augmented Generation

의미

RAG는 외부 지식 검색 결과를 LLM 입력 context에 결합해 답변 품질과 추적 가능성을 높이는 방식이다.

관련 Claims

RAG는 모델 내부 지식만 사용하는 방식보다 최신 정보 반영에 유리하다.
검색 품질은 최종 답변 품질에 직접적인 영향을 준다.
