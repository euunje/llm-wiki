1. LLM Provider 선택

사용 중인 로컬 LLM 런타임을 선택하면 endpoint 기본값을 제안합니다.

Ollama

로컬 Ollama 서버. API key 보통 불필요.
recommended local

LM Studio

LM Studio local server를 OpenAI-compatible API로 사용.
desktop

Custom

OpenAI-compatible endpoint를 직접 입력.
advanced
API key는 `.env`에 저장 후 화면/API 응답/로그에 다시 노출하지 않습니다.
