Expected Folder Layout

We will create the following layout to house LLM-Wiki services:

📂 [Vault Root]/
├── 📁 raw/            ← Inbox: drop your raw files (PDFs, PPTX) here
├── 📁 wiki/           ← Wiki output: AI-generated categorizations go here
└── 📁 schema/         ← Settings: AGENTS.md taxonomy configuration
