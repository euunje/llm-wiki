Model Context Protocol 연동

Expose your personal wiki knowledge to Claude Desktop, Cursor, or any MCP client. Our web app runs a standard stdio/SSE server.

1. Configure Claude Desktop

아래 설정 조각을 Claude Desktop 설정 파일에 복사하세요.

{
  "mcpServers": {
    "llm-wiki": {
      "command": "docker",
      "args": ["exec", "-i", "llm-wiki-app", "wiki", "mcp-stdio"]
    }
  }
}

File Location:
- macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
- Windows: %APPDATA%\Claude\claude_desktop_config.json

2. Tools Provided

  • list_wiki_pages: Retrieve list of files and their categories.
  • get_page_content: Retrieve the content of a specific page.
  • search_wiki: Find matching documents via semantic search index.
